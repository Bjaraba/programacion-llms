# Segmentación de clientes + predicción de gasto
Primero agrupa clientes en 3 segmentos con KMeans, luego predice el gasto mensual de cada uno
entrenando un modelo de regresión por segmento.

## Descripción del dataset

- age -> Edad del cliente
- income_usd -> Ingreso anual
- visits_per_month -> Visitas mensuales
- avg_basket_usd -> Ticket promedio por visita
- monthly_spend_usd -> Gasto mensual (target)
- loyalty_years -> Años como cliente fiel
