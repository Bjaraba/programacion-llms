## Dataset: Descrición del dataset 

- tenure_months -> Meses como cliente

- num_complaints -> Quejas registradas
- num_products -> Productos contratados
- last_login_days -> Días desde último login
- churn -> Etiqueta (0 = se queda, 1 = abandona) **Variable Objetivo**

- Separe features y target (churn) en train y test

- Entrene un RandomForestClassifier(n_estimators=100, random_state=42)

  - "probabilities": array con la probabilidad de churn (clase 1)
  - "accuracy": accuracy sobre el test set
  - "feature_importances": dict {nombre_feature: importancia}
